# homebrew-ockam

## Ockam CLI Installation

### Install Ockam

```
brew install --cask build-trust/ockam/ockam
```

### Upgrade Ockam

```
brew update && brew upgrade ockam
```

## Ockam Desktop App Installation

### Install Portals

```
brew install --cask build-trust/ockam/portals
```

### Upgrade Portals

```
brew update && brew upgrade portals
```
